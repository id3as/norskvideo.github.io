# Norsk Studio Website — Progress Tracker

## Goal
Build a modern, polished website for Norsk Studio by combining the best elements from five design iterations into a single consolidated version.

---

## Plan

### Design Decisions (agreed)
- **Foundation:** V3 (hybrid dark/light) as the base
- **Hero:** V2-style 2-column layout (text left, product UI right) with animated status dot
- **Theme:** Dark header/footer, light content sections — modern SaaS hybrid
- **Brand colors:** Cyan `#00A9E0` + Lime `#84BD00`
- **Content width:** ~800px for prose sections; full-width for hero and diagrams
- **Feature sections:** Alternating left/right rows (from V3) — replace 4-column grids
- **Flow diagram:** Keep V1's animated signal-flow diagram — strongest differentiator

### Sections (in order)
1. [ ] Hero — 2-column, value prop + product UI preview

3. [ ] Trust bar (logo / stats strip)
