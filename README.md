# Norsk Studio — Homepage Mockups

Design explorations for the Norsk Studio homepage. Each version represents a distinct visual direction.

## Versions

### V1 — Infrastructure
`v1-infrastructure/index.html`

Dark, developer-focused. Near-black background with IBM Plex Mono + Syne typefaces. Animated SVG signal-flow diagram. Closest in spirit to Linear and Vercel. Targets developers and technically sophisticated broadcast engineers.

### V2 — Broadcast Professional
`v2-broadcast/index.html`

Dark hero transitioning to light sections on scroll. Space Grotesk + DM Sans. Three-camera intermediate signal-flow diagram with CMAF output. Closest to Grabyo and Touchstream. Targets live event producers and broadcast teams.

### V3 — Modern Middle
`v3-modern/index.html`

Hybrid dark/light alternating sections. Outfit + JetBrains Mono. Most complex signal-flow diagram — three cameras through switcher, graphic overlay, and HTML ticker to CDN, Twitch, and YouTube simultaneously. Targets broadcast professionals and future creator economy segment.

### V4 — Longshot Light
`v4-longshot-light/index.html`

Inspired by [longshothq.com](https://longshothq.com). White background, editorial restraint. Text-based workflow lifecycle diagram as the centrepiece hero element. Alternating feature rows with small UI mockups. Geist + Geist Mono typefaces.

### V4b — Longshot Dark
`v4b-longshot-dark/index.html`

Dark version of V4. Full `#00263A` background throughout. Same Longshot-inspired structure — sparse hero, text-based workflow diagram, feature rows — but with the Norsk dark blue as the canvas. Light blue (`#00A9E0`) for accents and CTAs, green (`#84BD00`) for status indicators and capability checkmarks.
