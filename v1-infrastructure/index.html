How It Works

Build. Run. Connect.

Three steps from first workflow to production deployment.

01
Build

Design your live workflow visually using Norsk Studio's drag-and-drop interface. Connect sources, add encoding and packaging, insert graphics, set up source switching — no code required. Save any workflow as a reusable template.

RTMP · SRT · SDI · WebRTC
02
Run

Press play to deploy on your own hardware, in the cloud, or both. Norsk runs natively on AWS, GCP, and Akamai. Switch sources, update graphics, and adjust settings during a live event — all from the same UI.

AWS · GCP · AKAMAI · DOCKER
03
Connect

Control workflows programmatically via OpenAPI. Trigger source switches, update overlays, or start recordings with simple HTTP requests. Go deeper with the Norsk Engine SDK for fully custom media applications.

OpenAPI · TypeScript SDK
