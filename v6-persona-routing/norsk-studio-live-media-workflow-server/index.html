What Norsk Studio does

Everything you need for live video

One platform, from ingest to delivery — without stitching together a dozen tools.

🎥
Multi-source ingest
Capture multiple sources simultaneously — SRT, RTMP, SDI, and more — and encode to multiple formats in a single workflow.
🎛
Production enhancements
Subtitles and captions, multi-camera source switching, graphics overlay, multiview monitoring, and more — all in the visual builder.
📦
Package and deliver
Package as CMAF, HLS, or DASH and push to any CDN, YouTube, Twitch, or your own origin — in a single drag-and-drop workflow.
🖥
Visual workflow builder
Design, iterate, and deploy workflows graphically. No video engineering degree required to get a production up and running.
☁️
On-prem or cloud
Run Norsk Studio on your own hardware or deploy instantly on Google Cloud Platform, Akamai, or AWS — your infrastructure, your choice.
💾
Save and reuse workflows
Once you've built and tested a workflow, save it as a .nrsk file and reuse it across productions without rebuilding from scratch.
Installation

Get started in minutes

Installing Norsk Studio couldn't be easier. The only prerequisite is Docker Compose.

1

Download a trial licence

A free trial licence is required to run Norsk Studio. It gives you full access to all features with no time limit on development use.

Get a free trial licence
2

Run the installer

With Docker Compose installed, run this single command to pull and launch Norsk Studio:

$ bash <(curl -L https://norsk.video/studio/install)

Follow the quickstart guide to complete the first launch from Docker Compose.
